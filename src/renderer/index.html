              Setup Instructions
            

            
          

          

            

              
              Follow these steps to set up automatic death detection and scene switching in OBS
            


            
Enable OBS WebSocket

            
    
              
  1. Open OBS Studio
    2. 
              
  2. Go to Tools â†’ WebSocket Server Settings
    3. 
              
  3. Check "Enable WebSocket server"
    4. 
              
  4. Note the Port (default: 4455) and Password if set
    5. 
              
  5. Click OK
    6. 
            


            
Connect to OBS

            
    
              
  1. In this app, enter the Host (usually 127.0.0.1)
    2. 
              
  2. Enter the Port (default: 4455)
    3. 
              
  3. Enter the Password if you set one in OBS
    4. 
              
  4. Click "Connect" button
    5. 
              
  5. Status badge should turn Connected
    6. 
            


            
Configure Scenes

            
    
              
  1. Click "Refresh Scenes" to load your OBS scenes
    2. 
              
  2. Select Live Scene - your normal gameplay scene
    3. 
              
  3. Select Map Scene - scene to show when G key is pressed (optional)
    4. 
              
  4. Select Death Scene - scene to show when you die in Rust
    5. 
            


            
Select Monitor

            
    
              
  1. Choose your game monitor from the dropdown
    2. 
              
  2. This selection is saved automatically
    3. 
            


            
Start Screen Capture

            
    
              
  1. Click "Start Monitoring" button
    2. 
              
  2. Your screen will be captured for death detection
    3. 
              
  3. The video preview shows your selected monitor
    4. 
            


            
Select Death ROI (Region of Interest)

            
    
              
  1. Click "Select Death ROI" button
    2. 
              
  2. Click and drag over the "DEAD" text on the Rust death screen
    3. 
              
  3. Release to capture the template
    4. 
              
  4. Badge should show Captured
    5. 
              
  5. This is saved automatically - you only need to do this once!
    6. 
            


            
Adjust Detection Settings

            
    
              
  1. Use the Threshold slider to adjust sensitivity (default: 85%)
    2. 
              
  2. Higher = more strict (fewer false positives)
    3. 
              
  3. Lower = more sensitive (may trigger false positives)
    4. 
              
  4. Watch the Match Score badge while testing
    5. 
              
  5. Adjust Respawn Delay to prevent coordinate leaks (default: 200ms)
    6. 
            


            
How It Works

            
    
              
  • Death Detection: When you die in Rust, the app detects the "DEAD" text and switches to your Death scene. When you respawn, it automatically returns to Live scene after the configured delay.
    • 
              
  • G Key (Map): Press and hold G to switch to Map scene. Release G to return to Live scene after the configured delay. (G key is disabled while in Death scene)
    • 
              
  • Respawn Delay: Adds a buffer (default 200ms) before returning to Live scene. This prevents accidentally showing your coordinates/location when the game UI is still transitioning. The app may detect respawn faster than the game clears the screen.
    • 
              
  • Persistence: Your monitor, ROI, and template are saved automatically. On restart, everything loads and works immediately!
    • 
            


            

              
              Troubleshooting:
              
    
                
  • If death detection doesn't work, try lowering the threshold or re-capturing the ROI
    • 
                
  • Make sure the "DEAD" text is clearly visible when you capture the ROI
    • 
                
  • Check the Match Score badge to see detection accuracy in real-time
    • 
                
  • If G key doesn't work, make sure iohook badge shows Available
